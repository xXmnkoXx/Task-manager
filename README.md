## Introduction 
This application is a task management system designed to help users organize, manage, and share tasks efficiently. The interface provides features such as task creation, filtering, editing, and viewing public tasks from other users.

## My tasks
### In this window, we can see the tasks we have created and view their priority through colors, as well as filter by status.

![image](https://github.com/user-attachments/assets/011b34a2-e23f-4565-923c-857d1e8e0264)

## Task manager
### If we click on any of our tasks, we will move to an editing window where we can modify parameters or even delete the selected task.

![image](https://github.com/user-attachments/assets/e753045b-449d-4a27-81fd-4b97cd87594c)


## General tasks
### In this window, we can see "public-only" tasks from other users. The priority is also represented by colors, and we can filter tasks by their status.

![image](https://github.com/user-attachments/assets/55fe1aa5-aef4-4eba-98b5-3ecee5fbe2c7)
